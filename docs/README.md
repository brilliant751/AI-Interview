# AI-Interview 文档索引

更新时间：2026-06-28

本目录收纳项目交付、演示、开发、测试和运维需要的正式说明文档。根目录 `README.md` 保持为快速启动入口；本目录文档用于展开说明系统能力、业务流程、接口边界和运行维护细节。

## 推荐阅读顺序

1. [AI-Interview功能介绍.md](./AI-Interview功能介绍.md)：了解平台定位、功能模块、典型流程和技术栈。
2. [用户使用手册.md](./用户使用手册.md)：面向普通用户，说明注册、简历、岗位、面试、练习、报告等页面的使用方式。
3. [管理员使用手册.md](./管理员使用手册.md)：面向管理员，说明题库管理、知识库重建、provider 健康检查和数据材料维护。
4. [系统架构说明.md](./系统架构说明.md)：面向开发与评审，说明前端、后端、数据、模型 provider、异步任务和降级链路。
5. [API接口说明.md](./API接口说明.md)：按业务域整理主要接口、鉴权、幂等、错误响应和调试方式。
6. [数据与知识库说明.md](./数据与知识库说明.md)：说明 SQLite 表、材料目录、题库构建、向量库、报告文件和数据初始化。
7. [部署与运维手册.md](./部署与运维手册.md)：说明本地启动、环境变量、端口、provider 配置、备份恢复和故障排查。
8. [测试与质量说明.md](./测试与质量说明.md)：说明本地测试、CI 门禁、E2E、数据脚本 dry-run 和发布前检查。
9. [演示与验收指南.md](./演示与验收指南.md)：面向课程演示、评审和验收，提供演示脚本、检查清单和常见问题。
10. [业务流程详解.md](./业务流程详解.md)：按用户、管理员和系统视角说明注册、简历、JD、面试、报告、练习和知识库维护流程。
11. [数据库字段字典.md](./数据库字段字典.md)：按数据域说明主要表、字段含义、业务关系和排查路径。
12. [开发协作规范.md](./开发协作规范.md)：说明分支、提交、接口联调、测试、文档维护和代码 review 约定。
13. [验收与答辩材料.md](./验收与答辩材料.md)：整理课程验收、答辩讲解、演示脚本、检查表和常见问题。

## 文档维护约定

- 新增或修改接口时，同步更新 [API接口说明.md](./API接口说明.md)、`openapi/openapi.yaml` 和 Postman 集合。
- 新增页面或用户流程时，同步更新 [用户使用手册.md](./用户使用手册.md) 或 [管理员使用手册.md](./管理员使用手册.md)。
- 新增数据表、材料目录、导入脚本或构建报告时，同步更新 [数据与知识库说明.md](./数据与知识库说明.md)。
- 新增或修改核心表字段、状态流转和数据关系时，同步更新 [数据库字段字典.md](./数据库字段字典.md)。
- 修改启动、环境变量、provider、端口或依赖时，同步更新 [部署与运维手册.md](./部署与运维手册.md)。
- 修改测试脚本、CI、质量门禁或发布检查时，同步更新 [测试与质量说明.md](./测试与质量说明.md)。
- 修改演示主线、验收口径或答辩材料时，同步更新 [验收与答辩材料.md](./验收与答辩材料.md)。
